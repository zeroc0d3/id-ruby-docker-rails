STEP-03
CREATE DOCKER FILE 
----------------------------------------------
Simple Docker File:

FROM ruby:2.4.1-apline
MAINTAINER id-ruby
EXPOSE 80

Build Simple Docker: 
docker build --no-cache -t zeroc0d3/id-ruby-sample .

Running Simple Docker:
docker run -it --rm -p 80:80 zeroc0d3/id-ruby-sample
